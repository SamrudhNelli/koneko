# Koneko
A modern cross platform implementation of the popular Win95 cursor chasing cat.
* currently only supports hyprland systems

### Installation
Clone the repo using `git clone https://github.com/SamrudhNelli/koneko.git`\
Checkout the folder with `cd koneko`

### Usage
Run it using `./koneko.sh`\
Note : While running it for the first time, make sure to give the script executable permission using `chmod +x  koneko.sh`.
