# Koneko
A modern cross platform implementation of the popular Win95 cursor chasing cat.
* currently supports hyprland and windows

### Installation
Clone the repo using `git clone https://github.com/SamrudhNelli/koneko.git` or download the source code as a zip file.\
Checkout the folder with `cd koneko`

### Usage on Windows
Run it by double clicking on the `koneko.bat` script provided in the parent directory.\
If you are using shell enter `./koneko.bat` to execute the script.\
Note: Since the app is built with a GUI flag, you need to close it using the Task Manager.

### Usage on Linux
Run it using `./koneko.sh`\
Note : While running it for the first time, make sure to give the script executable permission using `chmod +x  koneko.sh`.
